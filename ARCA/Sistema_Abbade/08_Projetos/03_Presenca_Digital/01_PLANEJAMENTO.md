## 🎯 VISÃO DO PROJETO

### **O QUE É:**

```
Presença digital completa para Acib Informática:
1. Site institucional (serviços, sobre, contato)
2. Loja virtual (produtos: impressoras, tintas, acessórios)
3. Portfólio (serviços: manutenção, montagem, notebooks)
4. Integração com WhatsApp/Telegram
5. SEO (aparecer no Google quando buscarem)
```

### **OBJETIVO:**

```
Ter uma presença profissional na internet onde clientes podem:
• Ver serviços oferecidos
• Comprar produtos online
• Solicitar orçamentos
• Entrar em contato (WhatsApp, formulário)
• Ver localização/horários
```
