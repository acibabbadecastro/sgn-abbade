**Criado:** 2026-04-09 23:18  
**Status:** 📋 Planejamento Inicial  
**Prioridade:** ALTA

---

## 🎯 VISÃO DO PROJETO

### **O QUE É:**

```
Plataforma SaaS onde clientes podem:
1. Criar pedidos de sistemas de IA
2. IA especializada em fluxo de e-mails
3. Dashboard online para acompanhamento
4. Automação inteligente de inbox
```

### **OBJETIVO:**

```
Criar uma plataforma acessível para pequenas/médias empresas
gerenciarem e-mails automaticamente com IA, com:
• Triagem automática
• Respostas inteligentes
• Workflows personalizáveis
• Dashboard de métricas
```

## 🛠️ STACK TECNOLÓGICA SUGERIDA

### **BACKEND:**
```
🐍 Python + FastAPI
   • Bibliotecas de e-mail maduras
   • Fácil integração com IA
   • Rápido desenvolvimento
```

### **FRONTEND:**
```
⚛️ React ou Vue.js
   • Templates prontos
   • Muitos tutoriais
   • Componentes de dashboard
```

### **BANCO DE DADOS:**
```
🗄️ PostgreSQL (JÁ USA!)
   • Schema: emails, clientes, workflows, logs
   • Experiência prévia (SGN)
   • Robusto e confiável
```

### **IA:**
```
🤖 Desenvolvimento: Ollama (local, grátis)
   • Modelos: Llama 3, Qwen, Mistral
   
☁️ Produção: OpenAI/Anthropic (API)
   • Mais confiável para clientes
   • Escalável
   
🔧 Opcional: Fine-tune modelo próprio
```

### **INFRAESTRUTURA:**
```
🐳 Docker + Proxmox (JÁ TEM!)
   • Containers para cada serviço
   • Fácil backup/restore
   • Escalabilidade horizontal
```

### **PAGAMENTOS:**
```
💳 Mercado Pago / Stripe
   • Assinaturas recorrentes
   • Integração simples
```
