## 💡 DICAS DE VENDA

### Psicologia do pagamento:
```
✅ Pessoa acabou de receber → Dinheiro "extra" na cabeça
✅ Sexta-feira → Disposição para gastar
✅ Fim de semana chegando → Quer resolver rápido
✅ Mostrar produto pronto → Entrega imediata
```

### Frases que funcionam:
```
• "Entrega hoje mesmo!"
• "Pronto pra usar agora!"
• "Pagamento facilitado: Pix, cartão, dinheiro"
• "Garantia de 90 dias incluída"
• "Só tenho essa unidade"
```
