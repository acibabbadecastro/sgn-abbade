## 📦 SKILLS DISPONÍVEIS NO CLAWHUB

### COMO INSTALAR:

```bash
# Listar skills disponíveis
openclaw skills search <termo>

# Instalar skill
openclaw skills install <nome-da-skill>

# Atualizar skills
openclaw skills update

# Remover skill
openclaw skills remove <nome-da-skill>
```
