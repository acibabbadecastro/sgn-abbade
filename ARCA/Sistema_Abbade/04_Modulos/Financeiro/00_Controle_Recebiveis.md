## ⚠️ REGRAS DE COBRANÇA

### Política de Atraso:
```
• 1-7 dias: Lembrete amigável (WhatsApp/ligação)
• 8-15 dias: Cobrança formal
• 16-30 dias: Cobrança mais firma + juros
• 31+ dias: Considerar ações legais ou negativação
```

### Juros Aplicáveis:
```
• Até 30 dias: Sem juros (cortesia)
• 31-60 dias: 2% ao mês
• 61-90 dias: 5% ao mês
• 90+ dias: 10% ao mês + multa contratual
```

## 📞 MODELO DE MENSAGEM COBRANÇA

### Amigável (1-7 dias):
```
"Oi [Nome], tudo bem? Passando para lembrar que o pagamento de 
R$ [valor] referente a [serviço] venceu dia [data]. 
Pode me confirmar quando consegue fazer? Obrigado!"
```

### Formal (8-30 dias):
```
"[Nome], estou entrando em contato sobre o pagamento pendente 
de R$ [valor] (vencimento: [data]). Preciso que regularize o 
mais breve possível. Aguardo retorno."
```

### Urgente (31+ dias):
```
"[Nome], o pagamento de R$ [valor] está atrasado há [dias] dias. 
Preciso que regularize IMEDIATAMENTE para evitar juros e encaminhamento 
para setor jurídico. Aguardo até [data limite]."
```
